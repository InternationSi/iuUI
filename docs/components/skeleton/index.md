# Skeleton 骨架屏

:::demo 骨架屏

```vue
<template>
  <h4 style="padding:8px" >基本用法</h4 style="padding:8px"></h4>
  <i-skeleton></i-skeleton>
  <h4 style="padding:8px" >长度和数量可变</h4 style="padding:8px"></h4>
  <i-skeleton :length="500" :num="6"></i-skeleton>
</template>

```

:::

### API

|  参数  |   类型   | 默认 |     说明     |
| :----: | :------: | :--: | :----------: |
| length | `number` | 300  | 可选，文本值 |
|  num   | `number` |  4   | 可选，文本框 |
