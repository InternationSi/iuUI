# vultures-vue

## 文档地址

[组件库文档 ](http://si7578.gitee.io/vultures/)
### 组件库是基于vue3搭建的组件库

- 基础组件库
- 可视化组件库
- 流程组件库
  
### npm安装组件

```javascript
npm i vultures-vue -S
```

### 项目使用

```javascript
import vul from 'vultures-vue'


createApp(App).use(vul).mount('#app')
```
